== Description ==
Simple book review block plugin for a wordpress site

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/fgc_book_review` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
